Welcome to My Project.

By YuliWANG

This is a project for android application of Face Recognizing. For this application,we have three activities,of which one for the MainPage,one for test,and a third one for real-time tracking and recognizing. For more details,read thedocuments

Features

  1. Implement the algorithm of detecting and Recognizing faces using both EigenFace and FisherFace with the help of OpenCV.
  2. Capable of detecting the positoin of eye corner and iris center accurately,the goal of wich is achived with properly calculated binarization.
